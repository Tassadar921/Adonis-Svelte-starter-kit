# Adonis & Svelte starter kit

### Front : Svelte 5, Tailwind 4, Axios

### Back : Adonisjs 6, Redis, VineJS

### Databases : PostgreSQL

### ORM : Lucid

### Features

- Login
- Reset password
- Forgot password
- Friends
- Blocked users
- Multi languages (English & French natively, easy to add more languages)
- Multi themes (Light & Dark)

**In development**

- Account creation
- Google oauth
- Microsoft oauth
- Discord oauth
